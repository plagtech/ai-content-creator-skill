# AI Content Creator

**All-in-one content creation agent for [OpenClaw](https://openclaw.ai) / [ClawHub](https://clawhub.ai).**

Write blog posts, social media, newsletters, scripts, and product copy. Generate images, video, and voiceovers. Research topics and produce multimedia — all from a single workflow via x402 micropayments.

## Install

```bash
openclaw skills install ai-content-creator
```

Or browse on ClawHub: [clawhub.ai/plagtech/ai-content-creator-x402](https://clawhub.ai/plagtech/ai-content-creator-x402)
